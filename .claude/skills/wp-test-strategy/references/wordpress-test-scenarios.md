# WordPress Test Scenarios

Reviewers should suggest tests that match the failure mode.

## Plugin Scenarios

- Activation and uninstall effects
- Settings save and sanitize flow
- Hook-driven behavior

## REST Scenarios

- Permission denied responses
- Input validation failures
- Success response shape

## Block Scenarios

- Save/render behavior
- Attribute migration and deprecations
- Editor UI interactions

## WooCommerce Scenarios

- HPOS-safe order access
- Cart or checkout regression paths
- Payment and webhook edge cases
