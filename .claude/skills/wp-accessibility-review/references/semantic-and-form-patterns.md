# Semantic and Form Patterns

Use native HTML first.

## Prefer

- `<button>` for actions
- `<a>` for navigation
- `<label for>` or wrapped labels for form controls
- Real headings for structure
- Error messages associated with the field they describe

## Avoid

- Clickable `div` elements for primary actions
- Placeholder text as the only label
- ARIA labels where a visible label should exist
